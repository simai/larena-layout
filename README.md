# Larena Layout

Developer vertical slice: the shared admin collection layout defines heading, toolbar, content and pagination regions for Pages and Users. This does not claim a theme builder, full page builder, or production readiness.

Universal layout and page composition engine for public pages, admin pages, dashboards, forms, lists, detail pages, documentation pages and widgets in Larena.

This repository is a clean Larena pre-codegen baseline. Package implementation code has not started.

Canonical specifications are in `simai/larena-specs`.
